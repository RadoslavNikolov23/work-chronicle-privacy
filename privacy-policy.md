Privacy Policy  
Last updated: 2026

Work Chronicle App (“the Application”) is developed by Radoslav Nikolov (“the Developer”).  
This Privacy Policy explains how the Application handles user data.

## 1. Overview
Work Chronicle is an offline mobile application designed to help users manage and track their work shifts.  
The Application does **not** collect, store, transmit, or share any personal data.

## 2. No Internet Access
The Application does not require or use an internet connection.  
All features operate entirely offline.

## 3. Data Collection and Storage
The Application:
- does **not** collect personal information  
- does **not** track user activity  
- does **not** use analytics tools  
- does **not** use advertising SDKs  
- does **not** share data with third parties  

All information entered by the user (shift patterns, dates, schedules) is stored **locally on the device** and remains fully under the user’s control.

## 4. Local Storage
The Application uses a local database to save:
- shift schedules  
- user‑generated configurations  
- monthly hour reports  

This data never leaves the device and is not accessible to the Developer.

## 5. Permissions
The Application does not request access to:
- internet  
- contacts  
- location  
- camera  
- storage  
- device identifiers  
- background services  

No sensitive permissions are used.

## 6. Children’s Privacy
Since the Application does not collect any personal information, it is safe for users of all ages.

## 7. Security
Because no personal data is collected or transmitted, there is no risk of data interception or unauthorized access through the Application.

## 8. Changes to This Privacy Policy
The Developer may update this Privacy Policy if the Application’s functionality changes.  
Any updates will be published on the same page.
